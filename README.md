# hathor
Multi-agent system for generating galaxy formation hypotheses and creating relevant plots from RAMSES simulation data.

### Development Installation
```bash
# Navigate to the cloned repository
cd path/to/hathor

# Install in editable mode with development dependencies
pip install -e ".[dev]"

# Run tests
pytest
```

### Basic Installation (without dev tools)
```bash
cd hathor
pip install -e .
```

## State of this Project
This is a work in progress and is currently optimized for data from the MEGATRON simulations.
